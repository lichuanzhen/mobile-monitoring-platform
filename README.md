# mobile-monitoring-platform

> A Vue.js project

## Build Setup

```bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```

## 基本目录结构

### src

    api ---------------接口请求模块
    assets ---------------项目静态文件存放目录
    components ---------------公共组件存放目录
    page ---------------页面逻辑组件存放目录
    router ---------------项目路由
    store ---------------项目状态存放目录
    styles ---------------项目样式存放目录
    base ---------------全局 css
    mixin ---------------基准字体及算法
    reset ---------------重置样式
    variable ---------------基本样式
    App ---------------统一入口组件
    main ---------------统一入口 js
